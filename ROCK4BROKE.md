## 🤔 电吉他 ABC

### 问：电吉他如何发声？

- 电吉他通过拾音器和音箱发声。拾音器装有磁铁和线圈。磁铁磁化附近的琴弦钢材，琴弦振动时引起线圈内磁通量发生变化从而产生感应电动势和感应电流。这一信号再经过处理与放大，输出给音箱并播放，这就是电吉他的声音。

### 问：为什么不能给电吉他直接连耳机？

- 我们先来复习一下最大功率传输定理：

    `当负载阻抗与功率输出接口的除源阻抗共轭时（对于纯电阻电路则是相等），负载获得的功率最大。`

- 对于一个**从接口传出的信号**，若接口的除源阻抗（一般称为**输出阻抗**）较大，则**习惯上**称这个信号为**高阻抗信号**；反之则是**低阻抗信号**。传输某个信号的前后级接口应该具有共轭的除源阻抗以保证传输功率最大，这称为**阻抗匹配**。

- 一般情况下，电吉他设备链的阻抗匹配不会完全相同，至少量级接近。电吉他虽然可以作为一个交流电源使用，但是其电动势极低（数十毫伏），而内阻极高（数十万欧姆）。耳机作为负载，阻抗一般不超过 100 Ω ，这意味着耳机获得的功率几乎是 0。为此电吉他不能直接连耳机。

### 问：电吉他的音量旋钮有什么用？

- 它可以调节声音信号的强弱，读数越大声音越响。本质上是一个电位器，调节电吉他接口的输出电压。可以与箱头或增益单块的 GAIN 旋钮联合调节增益。

### 问：电吉他的音色旋钮有什么用？

- 它可以调节声音信号中高频成分的含量，读数越大声音越亮。本质上是一个可变低通滤波器，滤除输出信号的高频成分。可以与箱头的三段均衡或 TONE 旋钮联合调节音色。

### 问：如何给旋钮读数？

- 标准持琴姿势下眼睛看向旋钮，正对着你的刻度就是旋钮的读数；或者可以将旋钮拧到最大值或最小值，观察刻度"10"或刻度"0"的位置，那里就是"指针"。

### 问：对于多拾音器电吉他，不同拾音器采集的信号有什么不一样？

- 好问题。我们来复习一下琴弦振动的波动特征：琴弦的振动是一系列**驻波**的叠加，而驻波的频率有高有低，最低的称为**基频**，其他的驻波频率大概是基频的整数倍，称为**谐波**。谐波频率相比基频的倍数是谐波的**次数**。

- 拾音器越接近某一次谐波的波腹，该次谐波被采集到的强度越大。

- **琴颈拾音器**的位置比较接近**基频与低次谐波的波腹**，可以采集到厚实的**中低频**；而能够在**琴桥拾音器**处取得波腹的只可能是**高次谐波**，这意味着琴桥拾音器的声音更加**明亮尖锐而富有冲击力**。

- 使用无阻尼解的原因是，我懒得写欠阻尼解。不论是无阻尼解还是欠阻尼解，琴弦的振动都体现出驻波叠加的特征。既然我们只关心波节与波腹的位置分布，那么无阻尼解也可以用于说明问题。

### 问：单线圈拾音器和双线圈拾音器有什么区别？

- 单线圈拾音器的特性是，它容易受到环境中电磁场（如灯光、电脑）的干扰从而采集到噪音。双线圈拾音器通过两个线圈反向缠绕，可以有效抵消这种噪音。如果噪音过大，检查一下是否面对干扰源，或者尝试在房间的不同位置练习。这其实也构成了我不买 Stratocaster 和 Telecaster 的原因。Gibson 的特定型号 LP 支持将拾音器切换为单线圈工作（切单），这说明双线圈可以不用但不能没有。除了琴体厚而重，高把位弹起来不自在之外，LP 没有缺点。声音不够脆甜只是一种特性。

### 问：综合效果器音箱是什么意思？

- 你知道的，电吉他的信号需要处理才能形成音色，信号的处理流程就是**效果链**。

- 处理信号一般需要多种变换，如果一个效果器只能实现一种变换，则称为**单块效果器**；如果一个效果器能实现多种变换，则称为**综合效果器**。

- 在传统的由单块效果器组成的效果链中，串联顺序一般是这样的：

    `吉他 -> 调音表 -> 压缩器 -> 哇音踏板 -> 均衡 -> 增益 -> 调制 -> 延迟 -> 混响 -> 放大器 -> 音箱箱体`

- 现在我们考虑一种情况——把上述效果链中`均衡 -> 增益 -> 调制 -> 延迟 -> 混响 -> 放大器`部分塞进一个综合效果器里面，那么这个特殊的综合效果器就叫做**箱头**。

- 箱头的均衡和增益部分称为**前级**，用于塑造音色。它负责处理来自吉他的微弱信号，通过调节增益（GAIN）来决定音色的失真度和饱和度（从清澈到 crunch 再到高增益失真）；通过均衡（EQ，通常是三段：低频、中频、高频）来塑造音色的骨架（比如增强低频让声音更厚实，提升高频让声音更明亮）。

- 调制、延迟、混响部分称为**效果环路**，用于音色的空间装饰。它们为干瘪的原始信号添加空间感和趣味性。混响模拟在不同空间中的反射，制造深度；延迟制造回声或增加延音，增加层次感；调制效果则让声音流动、旋转起来。

- 放大器部分称为**后级**，用于改变信号的阻抗匹配特性。它接收来自前级或效果环路的信号，并将已经塑形完毕的**高阻抗、低功率信号**转换为**低阻抗、高功率的信号**，从而拥有足够的能量去驱动扬声器（喇叭）这种低阻抗负载。

- 再进一步考虑将箱头和箱体做成单台设备，那么这就是**综合效果器音箱**。这是一套完整的“一体化解决方案”。

- 核心组成部分与在 JOYO 音箱上的实现：
    前级（音色塑造核心）

    功能：负责处理来自吉他的微弱信号，通过增益决定音色的失真度（从清音到过载再到失真）；通过均衡塑造音色的骨架（调节低、中、高频）。

    在 JOYO 音箱上的实现：


    ```
    JA-01：提供清音、失真两个预设通道，无独立调节，是固化了的“快餐式”前级。

    JA-02：提供清音、过载两个通道，支持音色与增益调节，实现了前级的基础可调性。

    JAM BUDDY：提供清音、过载、失真三个通道，每个通道均可独立调节音量、音色、增益，构成了一个完整且可调的前级系统。

    JAM BUDDY II：在三通道基础上，增加了三段均衡，并支持14种箱头模拟，这相当于提供了14种不同风格的前级让你选择，实现了从前级到后级的完整音箱模拟。
    ```

    效果环路（空间装饰）

    功能：为干瘪的原始信号添加空间感和趣味性，主要包括调制、延迟、混响等。

    在 JOYO 音箱上的实现：

    ```
    JA-01 / JA-02：无此类效果。你的音色是“干”的，缺乏空间感。

    JAM BUDDY：内置延迟效果，你可以为音色添加回声，增加层次感。

    JAM BUDDY II：内置混响、延迟、调制等多种效果，你可以混合使用，制造出从房间到大厅的空间感、复杂的回声以及合唱、飘忽等移动效果。
    ```

    后级与系统功能（动力引擎与便利性）

    功能：将处理好的信号进行功率放大以驱动喇叭，并集成练习工具。

    在 JOYO 音箱上的实现：

    ``
    所有型号：均内置后级功放和喇叭，并支持耳机输出用于静音练习，这解决了电吉他高阻抗信号不能直推耳机的阻抗匹配问题。

    JA-02 / JAM BUDDY / JAM BUDDY II：支持蓝牙伴奏，方便跟着伴奏练习。

    JAM BUDDY II：集成了调音器、鼓机、Looper 功能，并将输出接口细分为连接其他吉他设备的 OUTPUT 和连接调音台或声卡的 LINE OUT，提供了完整的家庭练习、创作乃至小型演出的能力。
    ```
